## <<

: 원하는 자리 숫자 **만들기**

- 1<<3: 0000 1000
- 1<<5: 0010 0000

<br>

## &

: 특정 자리의 비트를 **확인**

- 10&1<<3

    0000 1010

    0000 1000

    ⇒ 결과: 0000 1000

- 비트가 1이라면 결과 값이 **0이 아닐 것**이고, 비트가 0이라면 결과 값이 0이 될 것이다.

<br>

## |

: 특정 자리수에 비트 **변경** (0→1)

- 10|1<<2

    0000 1010

    0000 0100

    ⇒ 결과: 0000 1110

- 다른 비트에는 영향을 주지 않고 원하는 자리에 비트 1을 추가 하였다.

<br>

## 비트 마스킹을 이용하여 순열 만들기

재귀 순열 코드

```java
perm(cnt){
	if(cnt==N) {//생성 완료 return}
	for(int i=0;i<N;i++){
		if(!isSelected[i]){  //1
			numbers[i] = input[i]
			isSelected[i] = true   //2
			perm(cnt+1)
			isSelected[i] = false  //2
		}
	}

}
```

0.   현재 뽑힌 원소들을 확인할 비트 타입의 flag 변수 추가

1. 상태 확인 → & 연산자 이용

    `(flag & 1<<i ) == 0`  ⇒ i 번째 자리의 숫자가 0이라면 (선택되지 않았다면)

2. flag 변경하기 → | 연산자 이용

    `flag | 1<<i` ⇒ i 번째 자리수를 1로 셋팅, i 번째 원소를 선택

    - 변수 자체의 값을 변경한 것이 아니기 때문에 (매개변수값을 수정해서 재귀에 태움)
    - **선택 해제에 대한 코드를 넣어 줄 필요가 없다**
